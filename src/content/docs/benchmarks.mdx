Knitting includes benchmarks under `bench/` to compare Node, Deno, and Bun.

## Run the suite

```bash
./run.sh
```

Results are written into `results/`.

## JSON output

```bash
./run.sh --json
```

JSON output is useful for plotting scripts under `graphs/`.

## Charts

Charts live under `src/assets/benchmark/` with `global/` for cross-runtime metrics and `runtime/` for runtime-specific plots.
