Use this page to check runtime requirements and install Knitting correctly for your JavaScript runtime.

## Requirements

- Node.js 22+
- Deno 2+
- Bun 1.0+

## Install

Knitting is published on npm as `knitting`. Latest release: `0.1.55`.

<Tabs>
  <TabItem label="npm"><Code code="npm install knitting" lang="bash" title={"npm.sh"} /></TabItem>
  <TabItem label="Deno"><Code code="deno add --npm knitting" lang="bash" title={"deno.sh"} mark={['--npm']} /></TabItem>
  <TabItem label="Bun"><Code code="bun add knitting" lang="bash" title={"bun.sh"} /></TabItem>
</Tabs>

:::note
Knitting is also published on JSR as `@vixeny/knitting` — the same code under a
scoped name. The npm package is the simplest path on every runtime, so prefer it
unless you specifically need the JSR entry.
:::
