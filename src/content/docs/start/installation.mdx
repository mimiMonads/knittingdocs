Use this page to check runtime requirements and install Knitting correctly for your JavaScript runtime.

## Requirements

- Node.js 22+
- Deno 2+
- Bun (recent stable)

## Install

Knitting is published on JSR:

<Tabs>
  <TabItem label="Deno"><Code code="deno add --npm jsr:@vixeny/knitting" lang="bash" title={"deno.sh"} mark={['--npm']} /></TabItem>
  <TabItem label="NPM"><Code code="npx jsr add @vixeny/knitting" lang="bash" title={"npm.sh"} /></TabItem>
  <TabItem label="Bun"><Code code="bunx jsr add @vixeny/knitting" lang="bash" title={"bun.sh"} /></TabItem>
</Tabs>

:::note
Knitting needs host-side package metadata, so use `--npm` when installing
through `deno` or `jsr`.

<Code code="jsr add --npm @vixeny/knitting" lang="bash" title={"jsr.sh"} mark={['--npm']}/>
:::
