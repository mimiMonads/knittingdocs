## Requirements

- Node.js 22+
- Deno 2+
- Bun (recent)

## Install

Knitting is published on JSR:

<Tabs>
  <TabItem label="Deno"><Code code="deno add --npm jsr:@vixeny/knitting"lang="bash" title={"deno.sh"} mark={['--npm']} /></TabItem>
  <TabItem label="NPM"><Code code="npx jsr add @vixeny/knitting" lang="bash" title={"npm.sh"} mark={['--npm']} /></TabItem>
  <TabItem label="Bun"><Code code="bunx jsr add @vixeny/knitting" lang="bash" title={"bun.sh"} mark={['--npm']} /></TabItem>
</Tabs>

:::note
That knitting needs a file on the host side, reason why we need to add `--npm` for `deno` and `jsr`

<Code code="jsr add --npm @vixeny/knitting" lang="bash" title={"jsr.sh"} mark={['--npm']}/>


:::
